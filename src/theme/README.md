# Theme

Centralized design tokens: colors, spacing, typography, borders, and related visual values for Blue Briefing’s public-safety look.

Screens and components must import theme values instead of hardcoding colors.

## Files

- `colors.ts` — palette tokens
- `spacing.ts` — spacing scale
- `typography.ts` — text styles
- `radius.ts` — corner radii
- `shadows.ts` — elevation styles
- `index.ts` — shared exports and layout constants
