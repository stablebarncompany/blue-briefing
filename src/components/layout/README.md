# Layout Components

App shell and navigation structure components, including sidebar (desktop), bottom tabs (mobile), screen containers, and page headers.

Preserve responsive behavior for web, iOS, and Android.

## Components

- `AppShell` — responsive shell composing sidebar or bottom nav
- `DesktopSidebar` — left navigation for wide layouts
- `MobileBottomNav` — bottom navigation for narrow layouts
- `TopBar` — product header with Agency Secure status
- `PageContainer` — padded scrollable page wrapper
- `NavTabButton` — shared tab trigger presentation
